# ALARM-CLOCK-CONSOLE-BASED
Console-based Python alarm clock for setting and managing multiple alarms with customizable alerts.
Features:  
- Set alarms  
- Snooze option  
- Custom labels  
- Sound alerts (cross-platform)

Technologies/Tools Used:  
- Python 3  
- Modules: datetime, time, threading, playsound

Steps to Install & Run:  
1. Clone the repository  
2. Open terminal and run: python alarm_clock.py

Instructions for Testing:  
- Manually test by setting alarms and checking sound and label output  
- Optional: Add unit tests for time parsing and alarm logic

Extra Description (optional):  
Built to explore Python threading, file I/O, and real-world utility in a console environment.
